# Canvasly 🎨

> **The open-source Figma alternative** – Design, prototype, and collaborate directly in your browser

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
[![Next.js](https://img.shields.io/badge/Next.js-13+-black)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-Ready-blue)](https://www.typescriptlang.org/)
[![Vercel](https://img.shields.io/badge/Deploy-Vercel-black)](https://vercel.com)

**Canvasly** is a powerful, open-source web-based design and prototyping platform built with Next.js. Create stunning UI designs, collaborate in real-time, and export production-ready code – all without leaving your browser.

![Canvasly Interface Preview](https://via.placeholder.com/800x400/1a1a1a/ffffff?text=Canvasly+Interface)

## 🌟 Why Canvasly?

- **🆓 Completely Free** – No subscriptions, no limits, open-source forever
- **🌐 Browser-Based** – No downloads required, works on any device
- **⚡ Lightning Fast** – Built with Next.js for optimal performance
- **🤝 Real-Time Collaboration** – Design together with your team
- **💻 Developer-Friendly** – Export clean React/HTML/CSS code

## ✨ Features

### 🎨 **Intuitive Canvas Editor**
- Infinite canvas with smooth pan and zoom
- Drag-and-drop components with smart guides
- Professional frames, shapes, and drawing tools
- Layer management with grouping and nesting

### ⌨️ **Familiar Shortcuts**
Figma-style keyboard shortcuts for maximum productivity:
- `R` → Rectangle tool
- `T` → Text tool  
- `F` → Frame tool
- `V` → Selection tool
- `Cmd/Ctrl + D` → Duplicate
- `Space + Drag` → Pan canvas

### 🧩 **Rich Component Library**
Pre-built, customizable components:
- Navigation bars and headers
- Hero sections and CTAs
- Forms and input fields
- Buttons and cards
- Charts and data visualization

### 🔤 **Advanced Typography**
- **1000+ Google Fonts** with instant preview
- Custom font upload support
- Granular controls for size, weight, spacing, and alignment
- Text styles and design system integration

### 🎯 **Extensive Icon Libraries**
- **Material Icons** (2000+ icons)
- **Feather Icons** (287 icons)
- **Heroicons** (460+ icons)
- Custom SVG icon upload

### 📱 **Responsive Design Tools**
- Live preview across desktop, tablet, and mobile
- Breakpoint management
- Auto-layout and constraints
- Device-specific design variants

### 📂 **Professional Project Management**
- Organized workspace with Home, Projects, Drafts, Teams
- Smart folder system with search and filtering
- Right-click context menus (rename, duplicate, delete, move)
- Version history and auto-save

### 👥 **Real-Time Collaboration**
- Multi-user editing with live cursors
- Comment system with threaded discussions
- Team permissions and sharing controls
- Activity feed and notifications

### 📤 **Powerful Export Options**
- **Images**: PNG, SVG, PDF with custom resolution
- **Code**: Clean React components, HTML/CSS
- **Design Tokens**: JSON export for design systems
- **Prototypes**: Interactive preview links

### 🌗 **Modern Interface**
- Clean, distraction-free dark and light themes
- Customizable workspace and panels
- Accessibility-first design
- Mobile-responsive interface

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ and npm/yarn/pnpm
- Modern browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/canvasly.git
   cd canvasly
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   # Add your configuration
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🌍 Deployment

### Deploy to Vercel (Recommended)
```bash
npm i -g vercel
vercel
```

### Deploy with Docker
```bash
docker build -t canvasly .
docker run -p 3000:3000 canvasly
```

### Manual Deployment
```bash
npm run build
npm run start
```
