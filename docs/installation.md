## 1. Clone the Repository


---

## 2. Install Dependencies

Using **npm**:

```bash
npm install
```

Or using **yarn**:

```bash
yarn install
```

Or with **pnpm**:

```bash
pnpm install
```

Or with **bun**:

```bash
bun install
```
